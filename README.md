# 阴阳师攻略编辑器

基于Vue3+ElementPlus实现的阴阳师攻略编辑器,旨在减少各大攻略博主和玩家的工作量。

用户可以通过点击简单的操作,快速编辑阴阳师的攻略。


## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with [ESLint](https://eslint.org/)

```sh
npm run lint
```
